# SPY 0DTE Dashboard v3.0 — 启动步骤

## 前置条件
- Python 3.12+
- Node.js v24+
- Redis (Windows build, placed in `infra/bin/redis-server.exe`)
- Longport API 密钥已配置在 `backend/.env`

## 启动顺序

### Step 1: 启动 Redis Server
```powershell
.\scripts\redis-start.bat
```
- 确认输出: `Ready to accept connections on port 6380`

### Step 2: 启动 Backend (FastAPI + Uvicorn)
```powershell
cd backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8001
```
- 确认输出: `Uvicorn running on http://0.0.0.0:8001`
- 健康检查: `curl http://localhost:8001/debug/persistence_status`

### Step 3: 启动 Frontend (Vite + React)
```powershell
cd frontend
npm run dev
```
- 确认输出: `Local: http://localhost:5173/`
- 打开浏览器访问 `http://localhost:5173`

## 关闭顺序
1. 先关闭 Frontend (Ctrl+C)
2. 再关闭 Backend (Ctrl+C)
3. 最后关闭 Redis (Ctrl+C)

## 常见问题
- **端口冲突 8001**: `netstat -ano | findstr :8001` 查找占用进程，`taskkill /f /pid <PID>` 终止
- **Redis 连接失败**: 确认 `redis-server.exe` 在 `infra/bin/` 目录下
- **API 数据为空**: 确认 `.env` 中的 Longport 密钥正确
